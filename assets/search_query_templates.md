# Search Query Templates

## 一、这份文件的作用

这份文件用于提供一套可复用的 YouTube 搜索词模板。

它不是固定词库，而是“搜索句型模板”。

这样后续你不管想搜：
- AI Agent
- Workflow
- Prompting
- Claude / Gemini / ChatGPT
- AI 工具
- 实战项目

都可以直接套模板组合搜索。

## 二、使用原则

### 1. 先选主题，再套模板
先明确你这一轮想搜什么：
- Agent
- Workflow
- Prompt
- Tool
- Project

### 2. 先判断这轮搜索意图
这轮搜索到底是为了：
- 学习
- 实操
- 方法
- 发文素材
- 综合

### 3. 同一主题至少用 2-3 组模板交叉搜
不要只搜一次。  
同一主题建议从不同角度搜，避免漏掉好内容。

## 十二、推荐使用方式

先搜：
- tutorial
- step by step
- from scratch

再搜：
- workflow
- implementation
- best practices

最后搜：
- advanced
- trends
- tools

### 方式二：按内容目标搜
#### 目标是学习
优先：
- tutorial
- implementation
- workflow
- best practices

#### 目标是发文
优先：
- advanced tips
- trends
- top tools
- comparison

#### 目标是两者兼顾
组合：
- tutorial + best practices
- workflow + advanced
- build + trends

## 十三、使用提醒

不要把所有搜索词一次性全跑。  
建议分轮次：

### 第一轮
找系统内容

### 第二轮
找实操内容

### 第三轮
找发文素材

## 十四、最终提醒

好的搜索模板，不是为了搜更多视频，  
而是为了更快搜到：

- 值得学的视频
- 值得拆的视频
- 值得沉淀成 skill 的视频
- 值得发出来的内容
